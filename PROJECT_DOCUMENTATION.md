# Project Documentation: LuhnLearn

## Overview
LuhnLearn is an AI-powered visual learning workspace that turns topics, notes, screenshots, and class images into interactive concept maps.

## Problem
Students learn from scattered materials such as class notes, screenshots, websites, YouTube videos, and AI chat answers. These materials are hard to organize, remember, and revise.

## Solution
LuhnLearn helps students create visual concept maps, add their own notes/images, study using quizzes and flashcards, and track revision progress.

## Key Features
- AI topic-to-map generation
- Interactive concept map
- Node summaries and explanations
- Notes and images as connected nodes
- Notes and images attached under concepts
- Image analysis using Gemini Vision
- Flashcards and quizzes
- Progress and revision tracking
- Save and export maps

## Tech Stack
- React / Vite
- React Flow
- Tailwind CSS
- Google Gemini API
- LocalStorage
